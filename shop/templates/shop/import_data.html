To import data, please upload a CSV or Excel file with the following columns: name_of_shop, name_of_owner, phone, district, subdistrict, suco, aldeia, dimension, kind_of_banner, latitude, longitude, kind_of_channel

The first row of your file should contain the column headers.

Some columns are required: name_of_shop, name_of_owner, dimension, kind_of_banner, latitude, longitude.

For now kind_of_banner only have two value:

  1. Small Banner
  2. Big banner with frame
